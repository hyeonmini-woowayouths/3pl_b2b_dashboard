# Logs Index

> 세션별 작업 로그. 최신이 위. 각 로그는 `YYYY-MM-DD-topic-slug.md` 형식.
> 새 로그 생성: `bash hyperagents/scripts/new-log.sh "<topic>"`

## 로그 작성 원칙

- **작성 시점**: 세션 종료 시 또는 중요 마일스톤 완료 시
- **길이**: 핵심만. 5-15줄 권장
- **구조**:
  - 컨텍스트 (무엇을 하려고 했는가)
  - 관찰 (무엇이 일어났는가 — 특히 의외의 것)
  - 판단 (왜 이렇게 했는가)
  - 남은 숙제 / 다음 에이전트에게 전달할 점
